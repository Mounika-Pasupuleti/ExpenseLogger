# 💰 Expense Logger Web Application

Expense Logger is a full-stack web application that helps users track their income and expenses efficiently. It provides a secure authentication system, allows users to add, edit, and delete transactions, and displays the current balance based on recorded financial activities.

This project was developed using the MERN stack and demonstrates practical implementation of user authentication, REST APIs, and database integration.

---

## 🚀 Features

- 🔐 User Registration and Login
- 🛡️ JWT-based Authentication
- ➕ Add Income and Expense Transactions
- ✏️ Edit Existing Transactions
- 🗑️ Delete Transactions
- 📊 Automatically Calculate Current Balance
- 📅 Store Transaction Date and Description
- 💾 Data Stored in MongoDB Atlas
- 📱 Responsive User Interface
